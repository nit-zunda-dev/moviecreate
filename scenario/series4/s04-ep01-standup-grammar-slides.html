シリーズ4 第1話

Standupでフリーズしたのだ

現場あるあるから学ぶ、応用できる英会話文法

対応シナリオ: scenario/series4/s04-ep01-standup-freeze.yaml

導入: 朝会で詰まる理由

  • 単語は分かっても、文の型が出てこない
  • 時制・前置詞・条件節が曖昧だと、応用が効かない
  • 今日は3フレーズを文法の再利用可能な型として覚える

型1: I'm working on ...（現在進行形）

I am + V-ing + on + 対象

  • 現在進行形は「今まさに取り組み中」を示す
  • on は「タスクに取り組んでいる」の前置詞

I'm working on fixing a bug in the login page.

I'm working on writing test cases.

I'm working on improving CI speed.

型2: I'm blocked by ...（受動態）

I am + blocked + by + 原因

  • 受動態で「自分ではなく、外的要因で止まっている」を表現
  • by の後ろに障害要因を置く

I'm blocked by an unstable auth server.

I'm blocked by a dependency update.

I'm currently blocked by production access issues.

型3: ETA is ... if ...（条件節）

ETA is + 時間 + if + 条件

  • if節で「条件つき見込み」を明示できる
  • 不確実性を伝えつつ、責任ある回答になる

ETA is tomorrow afternoon if the server recovers.

ETA is Friday if review is completed today.

応用マップ: 場面が変わっても型は同じ

レビュー会

I'm working on addressing your comments.

I'm blocked by unclear acceptance criteria.

障害対応

I'm working on identifying root cause.

ETA is one hour if rollback succeeds.

見積もり相談

I'm working on a rough estimate.

ETA is next sprint if scope stays fixed.

1on1

I'm working on improving communication.

I'm blocked by lack of domain knowledge.

ミニドリル: 日本語を英語に変える

  1. 「今はログ調査を進めています」
  2. 「本番DBへの権限不足で詰まっています」
  3. 「承認が通れば、今日中に終わります」

例: I'm working on investigating logs.

例: I'm blocked by missing production DB permissions.

例: ETA is today if the approval goes through.

第1話まとめ

覚えるのは単語より 文法の骨組み。

  • I'm working on ...（進行中の作業）
  • I'm blocked by ...（外的要因で停止）
  • ETA is ... if ...（条件つき完了見込み）
