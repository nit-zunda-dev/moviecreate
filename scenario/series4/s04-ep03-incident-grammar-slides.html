第3話

障害対応で頭が真っ白なのだ

現場あるあるから学ぶ、インシデント初動の短い型

対応シナリオ: scenario/series4/s04-ep03-incident-blank.yaml

導入: 初動で詰まる理由

  • 原因が未確定でも、沈黙は最悪。短い進行報告が必要
  • We でチームとして話すと、ウォールームで自然
  • 今日は3フレーズを状況 → 仮説 → 条件つき見込みの順で覚える

型1: We're ...ing now.（初動の進行）

We are + V-ing + now

  • 現在進行形で「今まさにやっている」を共有
  • now で切迫感と鮮度を添える

We're investigating now.

We're triaging errors now.

We're rolling back now.

型2: We believe ... is related to ...（仮説の渡し方）

We believe + this is related to + 要因

  • believe は断定より弱く、初動に向く
  • related to で関連づけを短く示す

We believe this is related to the latest deployment.

We believe this is related to a traffic spike.

We believe this is related to a third-party outage.

型3: ETA for mitigation is ... if ...（緩和の見込み）

ETA for mitigation is + だいたいの時間 + if + 条件

  • for mitigation で目的（被害抑止）を明示
  • if節で条件付きの見込みを渡す

ETA for mitigation is about 30 minutes if rollback succeeds.

ETA for mitigation is one hour if we can disable the feature flag.

ETA for mitigation is roughly 45 minutes if caching recovers.

応用マップ: 聞かれ方別

今何してる？

We're collecting logs now.

影響範囲は？

We believe impact is limited to checkout.

いつ収まる？

ETA for mitigation is about 20 minutes if cache warms up.

次の更新は？

We'll post another update in 15 minutes.

ミニドリル: 日本語を英語に変える

  1. 「今は原因を調査しています（チームで）」
  2. 「直近のリリースが関係している可能性が高いです」
  3. 「ロールバックが通れば、緩和の目安は30分くらいです」

例: We're investigating now.

例: We believe this is related to the latest deployment.

例: ETA for mitigation is about 30 minutes if rollback succeeds.

第3話まとめ

パニックでも、短い順番で返す。

  • We believe this is related to ...（仮説）
  • ETA for mitigation is ... if ...（条件つき見込み）
