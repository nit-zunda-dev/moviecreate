# ずんだもん アプリ開発を学びなおす！【アーキテクチャ】

## シリーズ概要

**ずんだもん アプリ開発を学びなおす！** のうち、**どういうアーキテクチャにするか**（構成・設計の考え方）を、めたん先生とずんだの掛け合いで「1テーマ2〜4分」にまとめるシリーズ。AI時代に重要な3つの視点のうち、**構成の選び方・設計の判断**に特化した全20回。

- **トーン**: 勉強動画でありつつ、ずんだの素朴な質問・ツッコミで堅くしすぎない。「設計の勘所がわかった」と思ってもらう。
- **尺**: 1本あたり 2〜4分。1テーマ1本で完結。
- **差別化**: めたん×ずんだの掛け合いで「このチャンネルで聞く」理由を作る。

### キャラ役割

| 役割 | キャラ | 役割詳細 |
|------|--------|----------|
| 先生 | 四国めたん（ノーマル or ツンツン） | 解説の主役。要点をはっきり話す。 |
| ずんだ | ずんだもん（ノーマル） | 相づち・質問・視聴者代弁の質問。 |
| ナレーション | 四国めたん（ささやき） | 冒頭のテーマ告知、重要用語の補足。 |

### バズらせる工夫

- **タイトル**: 「〇〇を1分で」「選び方の基準」などベネフィット＋短時間。
- **サムネ**: めたん先生＋ずんだもん＋キーワード1語（マイクロサービス / RAG など）。タグ: #アーキテクチャ #システム設計 #VOICEVOX
