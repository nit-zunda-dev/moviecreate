# ずんだもん アプリ開発を学びなおす！【何を作るか】

## シリーズ概要

**ずんだもん アプリ開発を学びなおす！** のうち、**何を作るか**を、**個人開発**に注目してまとめるシリーズ。めたん先生とずんだの掛け合いで「1テーマ2〜4分」。一人で作るときのスコープの切り方・アイデアの決め方・「やること・やらないこと」を扱い、**作るものの決め方**に特化した全20回。

- **対象**: 個人開発者・副業で作る人・趣味でサービスを作りたい人。チームではなく**一人で作る前提**の内容。
- **トーン**: 勉強動画でありつつ、ずんだの素朴な質問・ツッコミで堅くしすぎない。「個人開発で何を作るか決めやすくなった」と思ってもらう。
- **尺**: 1本あたり 2〜4分。1テーマ1本で完結。
- **差別化**: めたん×ずんだの掛け合いで「このチャンネルで聞く」理由を作る。

### キャラ役割

| 役割 | キャラ | 役割詳細 |
|------|--------|----------|
| 先生 | 四国めたん（ノーマル or ツンツン） | 解説の主役。要点をはっきり話す。 |
| ずんだ | ずんだもん（ノーマル） | 相づち・質問・視聴者代弁の質問。 |
| ナレーション | 四国めたん（ささやき） | 冒頭のテーマ告知、重要用語の補足。 |

### バズらせる工夫

- **タイトル**: 「〇〇を1分で」「決め方の基準」などベネフィット＋短時間。
- **サムネ**: めたん先生＋ずんだもん＋キーワード1語。タグ: #個人開発 #副業 #スコープ #VOICEVOX
