# タスク：導入シーンの背景画像作成

## 概要
第1話「ずんだもん、天地創造に口をはさむのだ！」の冒頭シーン（`intro-book`）で使用する背景画像を作成します。

## 詳細
- **シーン設定**: 古本屋でずんだもんが不思議な本を見つけるシーン。
- **必要な要素**:
  - 古びた本が並ぶ、少し雑然とした温かみのある古本屋の内部。
  - 中央のテーブルに置かれた、謎めいた光を放つ古本。
  - アニメ調（高品質な背景美術風）。
- **出力先**: `image/background/` フォルダ。
- **スクリプト更新**: `scenario/series1-episode01.yaml` の該当箇所の背景パスを更新する。
