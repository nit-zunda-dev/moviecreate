# シリーズ4 第2話 構成メモ

## 話数情報

- タイトル：レビューコメントが怖いのだ
- 目的：PRレビューで角が立たない依頼・提案・対応表明の英語3フレーズを定着させる
- 対応シナリオ：`scenario/series4/s04-ep02-pr-review-scary.yaml`
- 画面レイアウト：黒板背景（`global.videoFrame` 使用）

## 今回の持ち帰り3フレーズ

1. `Could you clarify ...?`
2. `I suggest ...`
3. `I'll address ...`

## 演出意図

- 曖昧で怖いレビューコメントに対して、**確認の型**で切り返す
- 自分がレビュー側のときは **提案の型** で伝える
- 最後に **対応する意志** を短く示して信頼を落とさない
- 次回導線として障害対応英語（`We're investigating now.` など）を予告
