スライド内の文字ベースの図（SVG）に加えて、以下の3つのシーンについて直感的でプロフェッショナルな図解画像を生成し、HTMLに埋め込みます。（文字を含まないクリーンなイラストを生成します）

2. **スライド5：XSSとCSRFの仕組みの違い**
