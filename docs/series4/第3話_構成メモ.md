# 第3話 構成メモ

## 話数情報

- タイトル：障害対応で頭が真っ白なのだ
- 目的：インシデント初動で状況・仮説・見込みを短く伝える英語3フレーズを定着させる
- 対応シナリオ：`scenario/series4/s04-ep03-incident-blank.yaml`
- 画面レイアウト：黒板背景（`global.videoFrame` 使用）

## 今回の持ち帰り3フレーズ

1. `We're investigating now.`
2. `We believe this is related to the latest deployment.`
3. `ETA for mitigation is about 30 minutes if rollback succeeds.`

## 演出意図

- ウォールームで聞かれる **What's the status?** に対し、まず **今やっていること** を現在進行形で共有する
- 断定を避けつつ、**仮説** は `We believe ... is related to ...` で短く渡す
- 第1話の `ETA is ... if ...` を **緩和（mitigation）** の文脈に載せ替え、インシデントでも同じ骨組みが使えることを見せる
- 次回導線として要件調整英語（`To confirm, you mean ...?` など）を予告
