# 📄 企画書：YouTubeアニメーションシリーズ
## 【完全攻略】ずんだもんとめたんの「情報処理安全確保支援士」絶対防衛戦線（全27回）

### 1. コンセプト（企画の狙い）
情報処理安全確保支援士試験（登録セキスペ）の膨大で難解な試験範囲を、ずんだもんと四国めたんの掛け合いを通じて**「面白く、分かりやすく、そして完全に抜け漏れなく」**解説するアニメーション動画シリーズ。
単なる知識の丸暗記（インプット）にとどまらず、最難関である午後試験を突破するための「現場の思考力・読解力（アウトプット）」までを動画内で育成し、視聴者を確実な合格へと導く。

### 2. ターゲット視聴者
*   情報処理安全確保支援士試験の受験予定者（特に午後試験で躓いている層）
*   Webアプリケーション開発やインフラ構築に携わり、セキュリティの基礎を固めたいエンジニア
*   ずんだもん・四国めたんの解説動画ファン（IT初学者）

### 3. キャラクター設定と役割
*   **ずんだもん（初心者＆被害者担当）：**
    *   セキュリティ意識が低く、毎回ハッカーの餌食になる。
    *   視聴者の「分からない」「長文読みたくない」という本音を代弁する。
*   **四国めたん（解説＆司令官担当）：**
    *   セキュリティのスペシャリスト。ずんだもんを厳しくも的確に指導する。
    *   「試験ではここが問われる！」という重要ポイントを容赦なく叩き込む。

---

### 4. シリーズ構成（全27回）

本シリーズは、基礎知識を網羅する「インプット編（全24回）」と、午後試験の思考力を鍛える「実践演習編（全3回）」の2部構成とする。

#### 🛡️ 【第1部：知識の武器庫編（インプット）】全24回
IPAの最新シラバスを極限まで分解し、6つのシーズン（各4回）で完全網羅する。

*   **Season 1：ハッカーの手口編**（マルウェア、DDoS、標的型攻撃など）
*   **Season 2：Webアプリの闇編**（SQLインジェクション、XSS、CSRFなど）
*   **Season 3：鉄壁のネットワーク構築編**（FW、WAF、VLAN、VPNなど）
*   **Season 4：暗号と認証の数学編**（PKI、ハッシュ、SAML、FIDOなど）
*   **Season 5：組織防衛とマネジメント編**（ISMS、CSIRT、インシデント対応など）
*   **Season 6：最新トレンド編**（DevSecOps、コンテナセキュリティ、BCP/DRなど）

#### ⚔️ 【第2部：午後試験の極意編（実践演習）】全3回（★追加提案部分）
インプットした知識を使い、午後試験特有の「罠」を突破するスキルを**TRPG（テーブルトークRPG）やシミュレーションゲームのような形式**で体感させる。

| 話数 | テーマ | 動画の展開・演出（バズるポイント） | 育成するスキル |
| :--- | :--- | :--- | :--- |
| **第25回** | **ポンコツ企業を救え！長文読解ブートキャンプ** | ずんだもんが「架空の企業（ガバガバなネットワーク図）」に派遣される設定。問題文の中に隠された「社員の危険な行動（パスワードの使い回し等）」という**フラグ（伏線）**を、めたんと一緒に推理ゲーム感覚で回収していく。 | **① 架空システムからの読解力** |
| **第26回** | **大人の事情シミュレーター！予算ゼロの防衛戦** | 「最新のWAFを入れたいけど予算がない！」「古いシステムは止められない！」という上司の無茶振りにずんだもんが絶望。制約条件（ビジネス要件）の中で、既存の設定変更だけで凌ぐ「現場のリアルな泥臭さ」を面白おかしく解説する。 | **② 制約条件に合わせた思考力** |
| **第27回** | **未知の脅威襲来！原理原則で立ち向かえ** | 見たこともない最新のサイバー攻撃が発生。ずんだもんがパニックになる中、めたんが「認証」「認可」「暗号化」といった**セキュリティの基本原則**に立ち返り、未知の脅威の正体を論理的に暴いていく。 | **③ 未知への応用力と推測力** |

### 5. バズを生むための演出・戦略

1.  **「失敗からの学び」フォーマットの徹底**
    毎回、冒頭でずんだもんが「システムを破壊される」「情報漏洩して賠償金を請求される」などの悲惨な（しかしIT現場あるあるな）インシデントを起こすキャッチーな導入にする。
2.  **図解の多用とビジュアル化**
    「通信のやり取り」や「暗号鍵の受け渡し」など、目に見えないIT技術を、キャラクターが荷物を運ぶようなアニメーションや、分かりやすい図解で表現する。
3.  **「試験あるある」のメタ発言**
    「この選択肢、絶対ひっかけなのだ！」「IPAはなんでこんな回りくどい言い方をするのよ！」など、受験者が共感する「試験への愚痴」をスパイスとして挟む。
